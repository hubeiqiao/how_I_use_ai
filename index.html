Use Cases: Graduate Student Edition

The very first domain where AI can truly deliver economical benefits is coding. Even if you don't know how to code, AI can do things for you. As long as a task can be finished by coding, you can try it.

Analyzing and Producing Documents

By using CLI tools, you can allow AI to analyze local folders directly - no more uploading files! These tools provide automatic context management, allowing AI to work continuously for several hours.

Place all your documents and papers in a folder to provide AI as context. For some assignments, it can produce work directly with the best performance.

CLI Tools Local File Access

Tip 1: Run multiple CLI tools at the same time and allow them to evaluate each other's outcomes.

Building Slide Decks

Once you have documents, send them to AI to generate slide decks for presentations. I unlocked this ability 7 months ago with Claude Sonnet 3.7. This September, I found a way to convert HTML files into exact 16:9 PDF presentations!

Try My Tool:

JoeSlides - AI-Powered Document to Presentation AI-Powered Slide Creation - Cursor Meetup Ottawa

Tip 2: AI-generated slide decks directly are terrible. Use code to generate them. Edit HTML instead of PowerPoint - AI tools aren't good at editing PPT yet.

Screening Papers

I'm researching AI for better English speaking and collected over 1,000 papers. For title/abstract screening, I used Codex CLI to build a custom tool. For full paper screening, Codex CLI analyzes local PDFs directly and updates results in CSV files.

My Tool:

Literature Screening Assistant

This opens a new way to speed up the literature review process.

Tip 3: LLMs can't access PDFs directly. Convert to markdown (.md) first. For tables, use .csv format - LLMs prefer these formats.

Generate Diagrams

In my research, I need several diagrams. In the past, I opened Figma to draw them manually. Now, I tell AI what I want and it generates Scalable Vector Graphics (SVG) files using coding abilities.

Then, it converts SVG to PNG/JPEG for immediate use. No more manual diagramming!

Building Prototypes for Yourself

The Most Exciting Use Case in 2025

Even if you don't know how to code, you can build things for yourself. As mentioned above, I built my literature screening tool because existing tools didn't meet my needs.

If you have a clear idea of what you want, try these tools and build one for yourself. For example, I'm exploring English speaking and building several tools I can use. It's way better than just describing in text.

While you might experience several issues, you'll at least have the experience and learn what to do next.

Video Generation

Finally Usable!

With Sora 2, you can use your own face to generate videos you want. Check out my video "Traveled The Anime Multiverse Using Sora 2!" where the entire content was generated by Sora 2.

With the Sora 2 API, you can generate and edit whole videos in just a few minutes. The only issue for me was my imagination 😂 (or what I wanted).

You can also use Google Veo 3.1 and Grok to generate videos.
